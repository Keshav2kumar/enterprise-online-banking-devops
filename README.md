# Enterprise Online Banking - DevOps Project

This repository contains a production-ready DevOps project template for an enterprise online banking application.
It demonstrates CI/CD (Jenkins), static code analysis (SonarQube), containerization (Docker), orchestration (Kubernetes),
artifact management (Nexus optional), monitoring (Prometheus + Grafana), logging (ELK), and infrastructure automation (Terraform).

Branches:
- develop → DEV
- release → QA
- master → PROD

Replace all placeholder values (REGISTRY_URL, K8S_NAMESPACE, DOMAIN_NAME, secrets) with your environment values.

See `docs/INSTALL.md` for quickstart and verification steps.
